 A plugin which Create a Google map by address. 



== Description ==

A plugin which Create a Google map by address where you can set your Marker by using shortcode [spyrowebz_gmap adress="your location address"]. 

Active your Plugin and then go to post or page where you want to show your google map. and use this shortcode [spyrowebz_gmap adress="your location address"].

Replace 'your location address' by your address fro example.

[spyrowebz_gmap address="Dir"]

where Dir is my address.

The beauty of plugin:-

You can use multiple google maps as much as you want just change your adress in your shortcode.

for example on my contact us page i use.

[spyrowebz_gmap address="Dir"]

on my about us page i use

[spyrowebz_gmap address="Karachi"]

This section describes how to install the plugin and get it working.

    2. Search for 'Spyrowebz Gmap'
    4. Install Spyrowebz Gmap
    3. Activate Spyrowebz Gmap from your Plugins page.
